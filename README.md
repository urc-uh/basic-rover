# Example ROS Project for a Basic Rover

This repository is a WIP that will contain an example ROS workspace that illustrates how programming to interface with a basic rover might look, further examples of using other ROS packages, and some tutorials to help you get started with ROS and our team workflow.

To properly work with this code you will need an installation of [ROS 2 Lyrical Luth](https://docs.ros.org/en/lyrical).
Start with [`ROS_INTRO.md`](ROS_INTRO.md) if you are new to ROS.
